# Greeting App

A simple Spring Boot web application that greets users by name using Thymeleaf.

## Features

- Enter your name in a form and get a personalized greeting.
- Built with Spring Boot and Thymeleaf.
- Simple and lightweight demo for learning Spring MVC concepts.

## Technologies Used

- Java 17
- Spring Boot 3.2.8
- Thymeleaf
- Maven
- HTML/CSS
